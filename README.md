# CSAI 320 Course Development Workspace

This workspace contains planning documents for the new course:

- [CSAI320-Overarching-Design.md](CSAI320-Overarching-Design.md)  
  Full course design document with program alignment, competencies, implementation strategy, and assessment architecture.

- [CSAI320-7-Week-Outline.md](CSAI320-7-Week-Outline.md)  
  Week-by-week thematic plan with activities, milestones, and evidence.

- [CSAI320-Rubric.md](CSAI320-Rubric.md)  
  Detailed grading rubric mapped to CLOs and competencies.

- [CSAI320-GitHub-Classroom-Implementation-Plan.md](CSAI320-GitHub-Classroom-Implementation-Plan.md)  
  Operational repository and CI strategy for assignment distribution and validation.

- [CSAI320-Weekly-Student-Instructor-Guides.md](CSAI320-Weekly-Student-Instructor-Guides.md)  
  Weekly instructions for students and instructor facilitation notes.

## Quick Start
1. Review the overarching design first.
2. Use the 7-week outline to sequence modules, labs, and grading artifacts.
3. Use the rubric to finalize grading criteria and feedback standards.
4. Convert weekly activities into GitHub Classroom starter repositories and CI workflows.
5. Deliver weekly student and instructor guides with each module release.

## Recommended Build Order
1. Finalize CLO to CC grading weights in the rubric.
2. Implement the GitHub Classroom repository templates and test harness.
3. Pilot Weeks 1 and 2 instructions with one internal teaching assistant.
4. Adjust pacing before full course launch.
